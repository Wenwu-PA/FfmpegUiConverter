# FFmpeg CLI Wrapper

## Описание

Терминальный интерфейс для ffmpeg, реализованный на C++.

## Возможности

- конвертация файлов
- история команд
- парсинг CLI-команд

## ООП

Использованы:

- инкапсуляция
- разделение ответственности
- композиция классов
